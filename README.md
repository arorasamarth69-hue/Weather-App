# Weather-App
This project uses OpenWeather API to create a weather app in which, the interface takes an input of the user collecting the name of the city and then in accordance to that, it searches for the temperature, wind-speed, weather, etc. and brings it. 
This project uses JQuery.
We use OpenWeatherApp's API Which you can generate by going on to:- 'https://home.openweathermap.org/'
